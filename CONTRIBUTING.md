# Contributing to Blue Arsenal

First off — thanks for considering contributing. Every detection, query, or cheatsheet improvement helps the whole DFIR community.

---

## What You Can Contribute

- 🔎 KQL hunting queries (Sentinel / Defender XDR)
- 🎯 Sigma rules (any platform)
- 🦠 YARA rules
- ⚙️ Triage scripts (PowerShell / Bash)
- 📄 Cheatsheet improvements
- ⛓️ New attack path documentation
- 🐛 Bug fixes / corrections

---

## Guidelines

### For Detection Rules (Sigma / KQL / YARA)
- Always include ATT&CK technique ID in comments or tags
- Include a brief description of what it detects and why
- Test before submitting — include false positive notes if any

### For Cheatsheets
- Keep it practical — field-usable, not theoretical
- Use tables and code blocks for readability
- Reference sources where applicable

### For Scripts
- Add a comment header explaining: purpose, OS, requirements
- Test on at least one environment before submitting

---

## How to Submit

1. Fork the repo
2. Create a branch: `feature/your-contribution-name`
3. Add your content
4. Open a Pull Request with a clear description

---

## Attribution

If your contribution is based on or inspired by external work, please credit the original source in the file.

---
